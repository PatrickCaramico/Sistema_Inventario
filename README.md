# Sistema de Inventario - Apresentacao

## Visao geral

Projeto web desenvolvido em PHP e MySQL para controle de estoque, com autenticacao por perfil, dashboard analitico, auditoria de acoes e assistente virtual integrado.

## Objetivo do projeto

Entregar uma aplicacao de inventario com foco em:

- seguranca de acesso
- usabilidade no dia a dia operacional
- leitura rapida de indicadores
- experiencia visual moderna
- diferenciacao para portfolio

## Principais destaques

- Login com controle de perfil
- Perfil Administrador e Perfil Funcionario
- Logout automatico por inatividade
- Dashboard com cards, insights e grafico por categoria
- Tabela com busca, filtros, ordenacao e paginacao
- Cadastro, edicao, exclusao e ajuste rapido de estoque
- Auditoria de acoes no sistema
- Assistente Luigi com respostas contextuais e perguntas sugeridas
- Assistente Luigi com respostas personalizadas citando o usuario logado

## Diferenciais

- Interface inspirada em mock visual de dashboard
- Modal de edicao com experiencia refinada
- Filtros de categoria com navegacao dedicada
- Chat de assistente virtual com interacao por clique e digitacao livre
- Separacao entre configuracao local e codigo versionado

## Tecnologias utilizadas

- PHP 8+
- MySQL
- PDO
- HTML5
- CSS3
- JavaScript
- Chart.js
